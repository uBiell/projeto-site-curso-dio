O que é a internet?

História

A história da internet teve início com a Arpanet, uma rede de troca de informações que havia sido desenvolvida para conectar instalações de pesquisas e militares com o Pentágono nos Estados Unidos, na década de 1960. O crescimento dessa rede permitiu que seu uso estendesse para fins privados e comerciais.

A internet é atualmente o maior meio de comunicação utilizado no planeta,com mais de 5 bilhões de usuários.Por ela, é possível enviar mensagens instâneamente, movimentar dinheiro em contas bancárias, comprar mercadorias, acessar redes sociais, fazer pesquisas etc. ¹

Client Side

O que são navegadores web?

Um navegador (também conhecido como web browser ou simplesmente browser) é um programa que habilita seus usuários a interagirem com documentos HTML hospedados em um servidor Web. A maior coleção interligada de documentos hipertexto, dos quais os documentos HTML são uma substancial fração, é conhecida como World Wide Web. ²

Aplicações Web

Qualquer programa que possa ser executado através do navegador web é uma aplicação web. Elas rodam em um computador conectado à internet e podem ser acessadas por qualquer outro computador conectado à internet, em qualquer lugar do mundo.

Logo, as aplicações web são desenvolvidas para serem executadas em computadores ou dispositivos móveis, como tablets e smartphones. Uma aplicação web pode variar desde simples páginas HTML até aplicações complexas que podem ser acessadas através de qualquer um dos mais populares navegadores do mercado. ³
